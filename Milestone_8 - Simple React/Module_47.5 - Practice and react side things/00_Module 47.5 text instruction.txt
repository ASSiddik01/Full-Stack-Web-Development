সাইড লাইনে বসে React বেসিক কনসেপ্ট ফকফকা করো (মডিউল ৪৭.৫) 

#module_release #simple_react_milestone #module_47_5

ধীরে ধীরে React এর জগতে তুমি প্রবেশ করতেছো। পরশু বেসিক কনসেপ্টগুলা এর সাথে প্রথম পরিচয় হয়েছে। তারপর গতকাল আবার সেই জিনিসগুলা সম্পর্কে জেনেছো। আরেকটা ডিফারেন্ট উদাহরণ দিয়ে। এরপরেও জিনিসগুলা সম্পর্কে আরেকটু কমফোর্টেবল হওয়ার দরকার আছে। কারণ এইগুলা React এপ্লিকেশন বানানোর মেইন জিনিস। অনেকটা জাভাস্ক্রিপ্ট এর মধ্যে ভেরিয়েবল, কন্ডিশন, array, লুপ, ফাংশন যেমন জাভাস্ক্রিপ্ট এ লাগে। তেমনি React এর মেইন মেইন জিনিসগুলা সব React এপ্লিকেশন এই লাগবে। 



.

React কোর কনসেপ্ট:

সেই জিনিসগুলো এর একটা লিস্ট আমি দিয়ে দিয়েছি। 

১. কিভাবে create react app দিয়ে নতুন React app তৈরি করবে 

২. কিভাবে কম্পোনেন্ট তৈরি করবে, এবং কম্পোনেন্ট ইউজ করবে

৩. props ইউজ করে কিভাবে কম্পোনেন্ট এ ডাটা সেন্ড করবে 

৪. destructuring ইউজ করে object এর প্রপার্টি ইউজ করবে 

৫. কিভাবে useState ইউজ করে স্টেট ডিক্লেয়ার করতে হয় 

৬. eventHandler দিয়ে কিভাবে স্টেট চেইঞ্জ করতে হয় 

৭. useEffect ইউজ করে কিভাবে API থেকে ডাটা লোড করতে হয় 

৮. লোড করা ডাটা কিভাবে দেখানো যায়  



এই জিনিসগুলোই আরেকটু ভালো করে দেখার দিন আজকে। 



.

আজকের ভিডিও: 

আজকে ৫ টা ভিডিও। একটু সাইডের জিনিস দেখানো হয়েছে। প্রথমেই হালকা করে ES6 Modules বিশেষ করে import এবং export কিভাবে কাজ করে সেটা একটু বলে দিয়েছি। তার সাথে সাথে relative path নিয়ে একটু ধারণা দেয়া হয়েছে। এরপর আছে কিভাবে তুমি Fake data তৈরি করতে পারবে, তোমার প্রজেক্ট এ সেই ডাটা লোড করতে পারবে। আর array এর reduce নিয়ে কিছু ধারণা। তারপর আছে Local storage নিয়ে কিছু কথা বার্তা আর সবার শেষে আছে UI কোনগুলা ভালো আর কোনগুলা খারাপ সেই নিয়ে কিছু আলোচনা। 



.



ইউটিউব থেকে রিভিশন:

তবে ভিডিও রিভিশন দিতে পারো বা না পারো। মেইন কথা হচ্ছে যে আটটা কনসেপ্ট এর লিস্ট দিয়েছি সেগুলা মোটামুটি বুঝতে হবে। তাহলেই কাজ শেষ। তবে আমি রিকমেন্ড করি তুমি ইউটিউব এ গিয়ে বাংলা বা ইংরেজিতে React এর টিউটোরিয়াল খুঁজে বের করে দেখো। অন্যদের ভিডিও থেকে শেখার ট্রাই করা অনেক বেশি ইম্পরট্যান্ট। এবং ভালো ডেভেলপার হওয়ার জন্য অনেক অনেক জিনিস ভিডিও থেকে ডকুমেন্টেশন থেকে শিখতে হবে। তাই এই জিনিসগুলা এখনই ধীরে ধীরে শুরু করে দেয়া উচিত।
